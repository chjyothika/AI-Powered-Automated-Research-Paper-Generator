# AI-Powered Scopus-Ready Research Paper Generator

## ⚠️ IMPORTANT DISCLAIMER
This tool assists in preparing submission-ready manuscripts. Human review, plagiarism checks, and peer review are REQUIRED before any submission.

## System Requirements
- Python 3.9 or 3.10
- NVIDIA GPU with CUDA support (RTX 4060 recommended)
- 8GB+ RAM
- Internet connection for arXiv access

## Features
- Direct arXiv API integration (HTTP + XML)
- GPU-accelerated AI generation
- Official publisher LaTeX templates (IEEE, Elsevier, ACM, Springer)
- RAG-based literature retrieval
- Section-wise paper generation
- Automatic BibTeX citation generation

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
streamlit run app.py
```

## Academic Ethics
- All generated content requires human verification
- Citations are auto-generated but must be validated
- Synthetic results are clearly labeled
- Proper attribution is maintained
